# ⚡ Tezkor Ishga Tushirish

## 1. Bot Token Oling
```
@BotFather → /newbot → Token oling
```

## 2. Admin ID Oling  
```
@userinfobot → /start → ID ni ko'ring
```

## 3. Sozlamalar
`.env` faylni oching va o'zgartiring:
```env
BOT_TOKEN=your_token_here
ADMINS=your_id_here
```

## 4. Kutubxonalar
```bash
pip install -r requirements.txt
```

## 5. Ishga Tushiring
```bash
python3 app.py
```

## ✅ Tayyor!
Botingiz endi ishlayapti! Telegram'da `/start` yuboring.
